About Our Project

Team Members: Jessilyn Collette, Liam Snow, Sarthak Swain, Yared Taye, Arsalan Saif

This project is a recommendation system that helps Github users discover new repositories by clustering their starred repositories into categories and providing personalized suggestions based on those categories. By using Github API and combining categorization techniques (k means) with recommendation algorithms such as Deterministic ranking algorithms, the project provides meaningful and targeted repository recommendations. This web-based application allows users to interactively explore relevant repositories based on their interests.
